지뢰 찾기

숫자 힌트를 바탕으로 지뢰가 없는 칸을 찾아내는 클래식 퍼즐 게임입니다. 지뢰를 클릭하지 않고 모든 안전한 칸을 열어보세요!

게임 규칙

  • 숫자는 주변 8칸의 지뢰 개수
  • 우클릭으로 지뢰 표시
  • 지뢰를 클릭하면 게임 오버
  • 모든 안전한 칸을 열면 승리

오목

흑백 돌을 번갈아 놓아 5개의 돌을 연속으로 놓는 전략 보드게임입니다. 상대방보다 먼저 5목을 완성해보세요!

게임 규칙

  • 흑돌(검은색)이 먼저 시작
  • 가로, 세로, 대각선 5목 완성
  • 금지된 수는 사용 불가
  • 5목을 먼저 완성하면 승리
